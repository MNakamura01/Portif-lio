Habilidades

Linguagens, ferramentas e frameworks que estudo.

HTML5

Linguagem de marcação utilizada na construção e estruturação de páginas na Web. Junto com o CSS3, é uma das tecnologias mais presentes no meu dia a dia.

JavaScript

A linguagem de programação mais famosa do mercado. O JS permite páginas da Web interativas e, portanto, é uma parte essencial dos aplicativos da web.

CSS3

Ferramenta responsável pela estilização de todo o site. A maior parte de tudo que você consegue ver é estilizado com CSS.

Bootstrap

Bootstrap é um framework para desenvolvimento de componentes de interface e front-end para sites e aplicações web. Utilizei Bootstrap pela primeira vez no curso Web Moderno.

Node.js

Node JS é um interpretador assíncrono de JavaScript, focando em migrar o JS do cliente para os servidores.

React.js

Um dos principais frameworks do JavaScript e o mais utilizado no mundo atualmente.

PHP

Linguagem de programação muito utilizada em backend.

Wordpress

Presente em mais de 30% de todos os sites, o Wordpress é um dos maiores e mais completos CMS que existe.

Git

Ferramenta de versionamento de código. Extremamente útil para organização e comunicação entre uma equipe bem estruturada.
